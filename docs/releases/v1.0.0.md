# CrabDesk 1.0.0

CrabDesk 1.0.0 是首个面向 Windows 10/11 x64 的正式版本。它将桌面盒子嵌入 Explorer 桌面层，不使用全局置顶窗口；普通应用和全屏窗口可以覆盖盒子，返回桌面后盒子仍然可见。

## 主要功能

- 创建、移动、缩放、折叠、重命名和删除桌面盒子。
- 盒子分组只保存元数据，不复制文件、不创建真实子目录，也不改变桌面文件路径。
- 未分组项目继续使用 Explorer 原生位置；项目移入盒子后隐藏对应的重复原生图标，移出或退出时恢复。
- 支持网格与列表视图、四种排序方式、图标尺寸与间距、标题样式、圆角、边框和选中颜色。
- 支持按项目类型、扩展名和名称通配符进行自动整理，并提供预览和批量撤销。
- 支持手动及每日布局备份、导入导出、保留策略和失败回滚。
- 支持文件夹映射盒子、只读模式、实时刷新以及离线重连。
- 支持全局快捷键、桌面右键菜单入口、双击桌面空白区域隐藏或显示图标。
- 支持浅色、深色和跟随系统主题，以及托盘、单实例和开机启动。
- 通过 GitHub Releases 检查稳定版或测试版更新；客户端不会保存 GitHub Token。

## 下载与校验

- `CrabDesk-Setup-x64.exe`：Windows x64 安装包。
- `CrabDesk-portable-win-x64.zip`：免安装便携版。
- `SHA256SUMS.txt`：发布文件的 SHA-256 校验值。

正式安装包应带有可验证的 Authenticode 签名。下载后请先核对数字签名和 `SHA256SUMS.txt`，再运行安装程序。

## 数据与升级

CrabDesk 配置保存在 `%LocalAppData%\CrabDesk`。安装或升级不会移动桌面文件，卸载默认保留布局配置和备份。建议升级前在“布局备份”页面创建一次手动备份。

退出或异常终止时，CrabDesk 与 IconGuard 会恢复接管前的 Explorer 图标显示状态和已记录的位置。

## 已知限制

- 首版仅支持 Windows 10/11 x64，不支持 Windows 7/8、ARM64 或虚拟桌面独立布局。
- 不包含云同步、自动下载安装更新、多标签组合盒子或真实文件夹整理规则。
- 已验证 Wallpaper Engine；其他动态壁纸软件、特殊 Explorer 修改工具和非标准桌面外壳仍需按兼容性清单逐项验证。
